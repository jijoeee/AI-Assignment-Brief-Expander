## 📌 The Cure for Procrastination Habit
A lightweight, powerful desktop application that transforms vague, 2-line assignment instructions into a highly structured, fully formatted academic roadmap in under 30 seconds. Built for students who need immediate direction.

### ✨ Key Features
* **🧠 Intelligent Structuring:** Generates section-by-section breakdowns, target word counts, and key points to cover.
* **📎 Context Injector (PDF Reading):** Upload your professor's syllabus or grading rubric (.pdf), and the AI will tailor the outline to match strict grading criteria.
* **🕸️ Visual Mind-Mapping:** Instantly converts your text outline into a stunning, interactive browser-based mind map using Mermaid.js.
* **📐 Perfect Format Export:** One-click export to `.docx` with auto-formatting for **APA 7th Edition** or **MLA 9th Edition** (including title pages, 12pt Times New Roman, and double spacing).
* **🔒 Secure BYOK (Bring Your Own Key):** Uses your own free Google Gemini API key, securely saved locally to your machine.

## 🛠️ Installation & Setup

**1. Clone the repository**
```bash
git clone [https://github.com/jijoeee/AI-Assignment-Brief-Expander.git](https://github.com/jijoeee/AI-Assignment-Brief-Expander.git)
cd AI-Assignment-Brief-Expander
```
2. Install dependencies
```bash
pip install -r requirements.txt
```
3. Run the application
```bash
python ai-assignment-brief.py
```
## 🔑 How to Get Your Free API Key
This app is powered by Google's blazing-fast Gemini 2.5 Flash model. You can use it completely for free.

Go to Google AI Studio.

Sign in with your Google account.

Click "Get API Key" and create a new key.

Open the app, click "⚙️ API Setting", and paste your key. You only have to do this once!

## ⚠️ Academic Disclaimer
This tool is designed to assist with planning, structuring, and brainstorming. It is NOT an essay writer. It creates a roadmap so you can write your own original content faster and with more confidence. Always adhere to your university's academic integrity guidelines regarding AI usage.
